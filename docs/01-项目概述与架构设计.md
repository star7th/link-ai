# 智链 AI 网关（LinkAI Gateway）- 项目概述与架构设计

## 一、项目概述

### 1.1 产品定位

智链 AI 网关是一款面向企业和个人用户的大模型安全接入网关，专注于调用过程中的**安全防护、数据脱敏、全链路审计与容灾保障**，让每一次 AI 调用都安全可追溯、稳定不掉线。

平台以**令牌路由**为核心机制，让团队管理者统一管控提供商资源、用量配额和安全策略，让每位成员在权限范围内自助创建令牌、选择模型提供商、配置容灾切换，兼顾安全管控与灵活使用。

### 1.2 核心价值

| 价值 | 说明 |
|------|------|
| 分层管控 | 管理员全局合规兜底，用户自助式使用，兼顾安全与效率 |
| 零改造接入 | 统一入口 + 令牌路由，业务/AI工具仅需替换 Token |
| 全能力闭环 | 代理转发 + 多用户管控 + 合规审计 + 数据脱敏 + 容灾自愈 + 用量管控一体化 |
| 合规可追溯 | 全链路不可篡改审计日志，全局强制脱敏规则，上游密钥零暴露 |

### 1.3 核心角色

| 角色 | 权限范围 |
|------|----------|
| 超级管理员 | 平台全局最高权限：用户管理、提供商资源池管理、全局安全策略、全量审计日志、配额管控、系统配置 |
| 普通用户 | 权限范围内自助使用：个人账号维护、自有令牌管理、查看开放提供商、配置容灾优先级、查看自有审计日志 |
| 用户组（可选） | 批量管理同部门/同角色用户，组内用户自动继承权限 |

## 三、系统架构设计

### 3.1 四层解耦架构

```
┌─────────────────────────────────────────────────────┐
│                   应用层 (Application)                │
│  ┌──────────────────┐  ┌──────────────────────────┐ │
│  │  管理员管控后台    │  │  普通用户自助控制台        │ │
│  └──────────────────┘  └──────────────────────────┘ │
├─────────────────────────────────────────────────────┤
│                 权限管控层 (Authorization)             │
│  用户管理 │ 角色权限 │ 开放范围控制 │ 配额校验 │ 强制规则 │
├─────────────────────────────────────────────────────┤
│                   引擎层 (Engine)                     │
│  代理转发 │ 脱敏规则 │ 容灾自愈 │ 审计日志 │ 配额管控  │
├─────────────────────────────────────────────────────┤
│                   资源层 (Resource)                   │
│         上游提供商资源池（统一配置、全局复用）             │
└─────────────────────────────────────────────────────┘
```

### 3.2 请求处理核心流程

```
客户端请求（携带平台令牌）
    │
    ▼
[1] 令牌认证 & 权限校验
    │ 验证令牌有效性、用户状态、IP黑白名单
    ▼
[2] 配额校验
    │ 检查用户配额、令牌配额是否充足
    ▼
[3] 脱敏处理（上行请求）
    │ 全局强制脱敏 → 用户自定义脱敏
    ▼
[4] 代理转发引擎
    │ 根据令牌绑定的提供商优先级，选择上游提供商
    ▼
[5] 容灾引擎（若请求失败）
    │ 自动熔断 → 无感切换备用提供商 → 探活自愈
    ▼
[6] 审计日志记录
    │ 记录完整请求/响应链路（脱敏后）
    ▼
[7] 配额扣减 & 返回响应
    │
    ▼
客户端收到响应（与原生调用体验一致）
```

│   ├── audit-logs/             # 我的审计日志
│   ├── usage/                  # 用量统计
│   ├── alerts/                 # 我的告警
│   └── profile/                # 个人信息
├── admin/                      # 管理员后台
│   ├── page.tsx                # 管理概览首页
│   ├── overview/               # 管理概览
│   ├── providers/              # 上游提供商管理

│   └── settings/               # 系统设置

│   ├── auth/[...nextauth]/     # NextAuth 认证路由
│   ├── auth/register/          # 用户注册
│   ├── proxy/[...path]/        # AI 代理转发核心路由
│   ├── admin/                  # 管理员 API
│   │   ├── users/              # 用户 CRUD
│   │   ├── providers/          # 提供商 CRUD
