## 一、核心技术实现

### 1.1 代理转发引擎

**位置**：`src/lib/proxy/engine.ts`

代理引擎是整个网关的核心，负责将客户端请求透传到上游提供商并返回响应。

```typescript
// 伪代码 - 代理转发核心流程
async function proxyRequest(req: Request, token: Token): Promise<Response> {
  const providers = getOrderedProviders(token.id); // 按优先级排序的提供商列表

  // 路径白名单校验：仅允许已知的 AI API 路径
  const requestPath = extractPath(req);
  if (!requestPath.startsWith('/v1/')) {
    return errorResponse(400, '不支持的代理路径');
  }

  for (const provider of providers) {
    // 检查熔断状态
    if (isCircuitOpen(provider.id)) continue;

    // 检查提供商配额
    if (!checkProviderQuota(provider.id)) continue;

    // 脱敏处理
    const sanitizedBody = applyDesensitize(req.body, token);

    // 协议适配
    const adapted = adaptRequest(provider.protocolType, sanitizedBody);

    try {
      // 转发请求
      const upstream = await fetch(provider.apiBaseUrl + path, {
        method: req.method,
        headers: adapted.headers,
        body: adapted.body,
      });

      if (upstream.ok) {
        // 成功处理
        if (isStreamRequest(req)) {
          return handleStreamResponse(upstream, provider, token);
        }
        recordAuditLog(token, provider, upstream, 'success');
        deductQuota(token, provider, upstream);
        return adaptResponse(provider.protocolType, upstream);
      } else {
        // 记录失败
        recordFailure(provider.id);
        continue; // 切换到下一个提供商
      }
    } catch (error) {
      recordFailure(provider.id);
      continue;
    }
  }

  return errorResponse(503, '所有上游提供商不可用');
}
```

### 1.2 流式响应代理

**位置**：`src/lib/proxy/stream.ts`

```
客户端 ←── SSE ──── 网关后端 ←── SSE ──── 上游提供商

实现方式：
1. 使用 ReadableStream 创建服务端流
2. 逐 chunk 从上游读取并透传给客户端
3. 流中断时向客户端发送 SSE error event，由客户端重试
4. 流结束时从最后一个 SSE data 中提取 usage 统计
```
客户端 ←── SSE ──── 网关后端 ←── SSE ──── 上游提供商

实现方式：
1. 使用 ReadableStream 创建服务端流
2. 逐 chunk 从上游读取并透传给客户端
3. 监听流中断事件，触发容灾续传
4. 流结束时从最后一个 SSE data 中提取 usage 统计
```

async function handleStreamResponse(
  upstream: Response,
  provider: Provider,
  token: Token
): Promise<Response> {
  const reader = upstream.body!.getReader();
  const encoder = new TextEncoder();

  const stream = new ReadableStream({
    async pull(controller) {
      const { done, value } = await reader.read();
      if (done) {
        // 流结束，统计 usage
        const usage = extractUsage(bufferedData);
        deductQuota(token, provider, usage);
        controller.close();
        return;
      }
      controller.enqueue(value);
    }
  });

  return new Response(stream, {
    headers: {
      'Content-Type': 'text/event-stream',
      'Cache-Control': 'no-cache',
      'Connection': 'keep-alive',
    },
  });
}

### 1.4 脱敏引擎

**位置**：`src/lib/desensitize/engine.ts`

```typescript
interface DesensitizeResult {
  processed: boolean;      // 是否命中规则
  content: string;         // 处理后的内容
  action: string;          // replace | block | log
  hits: DesensitizeHit[];  // 命中记录
}

function applyDesensitize(content: string, rules: DesensitizeRule[]): DesensitizeResult {
  let result = content;
  const hits: DesensitizeHit[] = [];

  for (const rule of rules.sort((a, b) => b.priority - a.priority)) {
    if (!rule.isEnabled) continue;

    if (rule.ruleType === 'keyword') {
      // 关键字精确匹配
      if (result.includes(rule.pattern)) {
        result = result.replaceAll(rule.pattern, rule.replacement || '[REDACTED]');
        hits.push({ ruleId: rule.id, match: rule.pattern, action: rule.action });
      }
    } else if (rule.ruleType === 'regex') {
      // 正则匹配
      const regex = new RegExp(rule.pattern, 'g');
      if (regex.test(result)) {
        result = result.replace(regex, rule.replacement || '[REDACTED]');
        hits.push({ ruleId: rule.id, match: rule.pattern, action: rule.action });
      }
    }

    if (rule.action === 'block' && hits.length > 0) {
      return { processed: true, content: '', action: 'block', hits };
    }
  }

  return { processed: hits.length > 0, content: result, action: 'pass', hits };
}
```

### 1.5 熔断器

**位置**：`src/lib/failover/circuit-breaker.ts`

基于内存的三态熔断器（单进程内 Map 管理，无需 Redis）：

```typescript
enum CircuitState { CLOSED, OPEN, HALF_OPEN }

interface CircuitRecord {
  state: CircuitState;
  since: number;
  successWindow: number[];  // 滑动窗口：请求成功时间戳
  failWindow: number[];     // 滑动窗口：请求失败时间戳
}

class CircuitBreaker {
  private circuits: Map<string, CircuitRecord> = new Map();
  private config: FailoverConfig;

  recordSuccess(providerId: string): void {
    const record = this.getOrCreate(providerId);
    record.successWindow.push(Date.now());
    this.trimWindow(record.successWindow);
  }

  recordFailure(providerId: string): void {
    const record = this.getOrCreate(providerId);
    record.failWindow.push(Date.now());
    this.trimWindow(record.failWindow);

    const total = record.failWindow.length + record.successWindow.length;
    if (total >= this.config.minRequestCount) {
      const errorRate = (record.failWindow.length / total) * 100;
      if (errorRate >= this.config.errorThresholdPercent) {
        this.trip(providerId);
      }
    }
  }

  isAvailable(providerId: string): boolean {
    const record = this.circuits.get(providerId);
    if (!record) return true;
    if (record.state === CircuitState.OPEN) {
      // 冷却期后自动转为半开
      if (Date.now() - record.since > this.config.cooldownSeconds * 1000) {
        record.state = CircuitState.HALF_OPEN;
        return true;
      }
      return false;
    }
    return true;
  }

  private trip(providerId: string): void {
    const record = this.getOrCreate(providerId);
    record.state = CircuitState.OPEN;
    record.since = Date.now();
  }

  private trimWindow(window: number[]): void {
    const cutoff = Date.now() - this.config.windowSeconds * 1000;
    while (window.length > 0 && window[0] < cutoff) window.shift();
  }
}
```

### 1.6 限流引擎

**位置**：`src/lib/quota/rate-limiter.ts`

基于内存滑动窗口的限流器，支持令牌级和提供商级 RPM/TPM 限制：

```typescript
class RateLimiter {
  private windows: Map<string, number[]> = new Map();

  checkAndRecord(params: {
    type: 'token' | 'provider';
    refId: string;
    limitType: 'rpm' | 'tpm';
    limit: number;
    cost: number; // 请求数=1（RPM），Token数（TPM）
  }): { allowed: boolean; remaining: number } {
    const key = `ratelimit:${params.type}:${params.refId}:${params.limitType}`;
    const now = Date.now();
    const windowMs = 60_000; // 1 分钟窗口

    let window = this.windows.get(key) || [];
    // 清除过期条目
    window = window.filter(ts => ts > now - windowMs);

    if (params.limitType === 'rpm') {
      // RPM：每个时间戳代表一次请求
      if (window.length >= params.limit) {
        this.windows.set(key, window);
        return { allowed: false, remaining: 0 };
      }
      window.push(now);
      this.windows.set(key, window);
      return { allowed: true, remaining: params.limit - window.length };
    } else {
      // TPM：每个时间戳附带 Token 消耗，用数组长度近似（简化方案）
      // 更精确方案：存储 { ts: number, cost: number }[] 并求和
      const totalCost = window.length; // 简化：每条记录视为 1 unit
      if (totalCost >= params.limit) {
        this.windows.set(key, window);
        return { allowed: false, remaining: 0 };
      }
      window.push(now);
      this.windows.set(key, window);
      return { allowed: true, remaining: params.limit - totalCost };
    }
  }

  // 定期清理过期窗口，防止内存泄漏
  startCleanup(): void {
    setInterval(() => {
      const now = Date.now();
      for (const [key, window] of this.windows) {
        const cleaned = window.filter(ts => ts > now - 60_000);
        if (cleaned.length === 0) {
          this.windows.delete(key);
        } else {
          this.windows.set(key, cleaned);
        }
      }
    }, 30_000);
  }
}
```

class HealthChecker {
  private intervalId: NodeJS.Timeout;

  start(config: FailoverConfig): void {
    this.intervalId = setInterval(async () => {
      const providers = await getActiveProviders();
      for (const provider of providers) {
        if (!config.healthCheckEnabled) continue;
        await this.checkProvider(provider);
      }
    }, config.healthCheckInterval * 1000);
  }

  private async checkProvider(provider: Provider): Promise<void> {
    const start = Date.now();
    try {
      // 轻量级探测：获取模型列表或发送最小化请求
      const adapter = getAdapter(provider.protocolType);
      const response = await fetch(provider.apiBaseUrl + '/v1/models', {
        headers: adapter.getAuthHeaders(provider.apiKey),
        signal: AbortSignal.timeout(config.healthCheckTimeout * 1000),
      });

      await recordHealthLog(provider.id, 'active', response.ok ? 'success' : 'failure',
        Date.now() - start, response.ok ? null : `HTTP ${response.status}`);
    } catch (error) {
      await recordHealthLog(provider.id, 'active', 'timeout',
        Date.now() - start, error.message);
    }
  }
}

### 1.8 审计日志引擎

**位置**：`src/lib/audit/logger.ts`

审计日志引擎支持可选的完整请求/响应体记录，通过 `SystemConfig` 的 `audit_log_full_body` 键控制开关。

```typescript
type AuditLogEntry = {
  userId?: string;
  tokenId?: string;
  providerId?: string;
  logType: string;
  action: string;
  requestMethod?: string;
  responseStatus?: number;
  responseTime?: number;
  promptTokens?: number;
  completionTokens?: number;
  totalTokens?: number;
  isStream?: boolean;
  failover?: boolean;
  ipAddress?: string;
  userAgent?: string;
  detail?: string;
  requestBody?: string;   // 完整请求体（需开启开关）
  responseBody?: string;  // 完整响应体（需开启开关）
};

class AuditLogger {
  private buffer: AuditLogEntry[] = [];
  private lastHash: string = '';

  log(entry: AuditLogEntry): void {
    // 计算链式哈希
    const contentHash = computeHash(this.lastHash + JSON.stringify(entry));
    entry.contentHash = contentHash;
    entry.previousHash = this.lastHash;
    this.lastHash = contentHash;

    // 写入内存缓冲区
    this.buffer.push(entry);

    // 缓冲区满100条时立即落库
    if (this.buffer.length >= 100) {
      this.flush();
    }
  }

  // 批量落库
  async flush(): Promise<void> {
    if (this.buffer.length === 0) return;
    const entries = this.buffer.splice(0, 100);
    try {
      await prisma.auditLog.createMany({ data: entries });
    } catch (error) {
      this.buffer.unshift(...entries);
    }
  }
}

    const providerUsed = this.usageMap.get(`provider:${params.providerId}:${period}`) || 0;

    this.usageMap.set(`token:${params.tokenId}:${period}`, tokenUsed + params.tokens);
    this.usageMap.set(`user:${params.userId}:${period}`, userUsed + params.tokens);

          where: { type_refId_period: { type, refId, period } },
          update: { usedTokens: value },
          create: { type, refId, period, usedTokens: value },

class AlertEngine {
  private cooldownMap: Map<string, number> = new Map(); // ruleId → 上次告警时间戳

  async evaluate(rule: AlertRule, context: Record<string, any>): Promise<void> {
    // 检查冷却期
    const lastAlert = this.cooldownMap.get(rule.id);
    if (lastAlert && Date.now() - lastAlert < rule.cooldown * 1000) return;

    // 渲染消息模板
    const message = renderTemplate(rule.messageTemplate, context);

    // 发送到各渠道
    for (const channel of rule.channels) {
      await this.sendToChannel(channel, rule, message);
    }

    // 记录告警日志
    await prisma.alertLog.create({ data: { ruleId: rule.id, message, ... } });

    // 设置冷却
    this.cooldownMap.set(rule.id, Date.now());
  }
}

```typescript
import { NextResponse } from 'next/server';
import type { NextRequest } from 'next/server';
import { getToken } from 'next-auth/jwt';

export async function middleware(request: NextRequest) {
  const { pathname } = request.nextUrl;

  // 1. 代理转发路由：令牌认证
  if (pathname.startsWith('/api/proxy/')) {
    const authHeader = request.headers.get('authorization');
    if (!authHeader?.startsWith('Bearer lk-')) {
      return NextResponse.json({ success: false, error: 'Missing or invalid token' }, { status: 401 });
    }
    // 令牌验证在 route handler 中完成（需要查询数据库）
    return NextResponse.next();
  }

  // 2. 管理员路由：Session + Admin 校验
  if (pathname.startsWith('/api/admin/') || pathname.startsWith('/admin/')) {
    const session = await getToken({ req: request });
    if (!session) {
      if (pathname.startsWith('/api/')) {
        return NextResponse.json({ success: false, error: 'Unauthorized' }, { status: 401 });
      }
      return NextResponse.redirect(new URL('/auth/login', request.url));
    }
    if (!session.isAdmin) {
      if (pathname.startsWith('/api/')) {
        return NextResponse.json({ success: false, error: 'Forbidden' }, { status: 403 });
      }
      return NextResponse.redirect(new URL('/dashboard', request.url));
    }
  }

  // 3. 用户路由：Session 校验（dashboard 为用户控制台入口）
  if (pathname.startsWith('/api/user/') || pathname.startsWith('/dashboard')) {
    const session = await getToken({ req: request });
    if (!session) {
      if (pathname.startsWith('/api/')) {
        return NextResponse.json({ success: false, error: 'Unauthorized' }, { status: 401 });
      }
      return NextResponse.redirect(new URL('/auth/login', request.url));
    }
  }

  return NextResponse.next();
}

    '/api/proxy/:path*',
    '/api/admin/:path*',
    '/api/user/:path*',
    '/admin/:path*',

- **逻辑**：加载所有启用的告警规则，评估触发条件，发送告警

## 四、安全实现

### 4.1 API Key 加密方案

```typescript
import { createCipheriv, createDecipheriv, randomBytes, scryptSync } from 'crypto';

const ALGORITHM = 'aes-256-gcm';
const IV_LENGTH = 16;
const TAG_LENGTH = 16;

function getEncryptionKey(): Buffer {
  const secret = process.env.NEXTAUTH_SECRET!;
  return scryptSync(secret, 'link-ai-salt', 32);
}

function encrypt(plaintext: string): string {
  const key = getEncryptionKey();
  const iv = randomBytes(IV_LENGTH);
  const cipher = createCipheriv(ALGORITHM, key, iv);
  const encrypted = Buffer.concat([cipher.update(plaintext, 'utf8'), cipher.final()]);
  const tag = cipher.getAuthTag();
  return Buffer.concat([iv, tag, encrypted]).toString('base64');
}

function decrypt(ciphertext: string): string {
  const key = getEncryptionKey();
  const data = Buffer.from(ciphertext, 'base64');
  const iv = data.subarray(0, IV_LENGTH);
  const tag = data.subarray(IV_LENGTH, IV_LENGTH + TAG_LENGTH);
  const encrypted = data.subarray(IV_LENGTH + TAG_LENGTH);
  const decipher = createDecipheriv(ALGORITHM, key, iv);
  decipher.setAuthTag(tag);
  return decipher.update(encrypted) + decipher.final('utf8');
}
```

function generateTokenKey(): string {
  const bytes = randomBytes(24);

// 哈希令牌密钥（用于认证查询）
function hashTokenKey(key: string): string {
  return createHash('sha256').update(key).digest('hex');
